Exercice 1 : Big Data & High Dimension

Guillaume Ertel - M2 NET

Lundi 16/01/2023

Thème - peintres

Code de la requête Big Data : 

            
                SELECT ?painterLabel ?point_in_time ?date_of_death ?movementLabel WHERE {
                SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE],en". }
                ?painter wdt:P106 wd:Q1028181.
                ?painter wdt:P569 ?point_in_time. 
                ?painter wdt:P570 ?date_of_death. 
                ?painter wdt:P135 ?movement. 
                }
                LIMIT 2000

Code de la requête High Dimension : 

            
                SELECT ?painterLabel ?point_in_time ?date_of_death ?movementLabel ?sex_or_genderLabel ?given_nameLabel ? 
                work_locationLabel ?country_of_citizenshipLabel ?artist_files_atLabel ?member_ofLabel ?image WHERE {
                SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE],en". }
                ?painter wdt:P106 wd:Q1028181;
                    wdt:P569 ?point_in_time;
                    wdt:P570 ?date_of_death;
                    wdt:P135 ?movement.
                    ?painter wdt:P21 ?sex_or_gender.
                    ?painter wdt:P735 ?given_name.
                    ?painter wdt:P937 ?work_location.
                    ?painter wdt:P27 ?country_of_citizenship.
                    ?painter wdt:P9493 ?artist_files_at.
                    ?painter wdt:P463 ?member_of.
                    ?painter wdt:P18 ?image.
                }
                LIMIT 100
